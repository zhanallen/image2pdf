# Image2PDF
**這是一個把圖片輕易轉換成PDF檔的小程式**  
支援以下模式

## UI 模式
* 支援同時轉換多張照片
* 可批量轉換圖片至PDF

## 快速模式
如要使用快速模式，直接將圖片拖放至應用程式圖標即可
* 支援同時多張照片合成單一檔案

---
### ***----- 備註 -----***
1. 快速模式下，PDF會儲存至程式執行的資料夾
2. 快速模式下，多張圖片的順序是依照檔名進行排序，如需手動排序，請使用 UI 模式
3. win10 部分 UI 支援度還需改良
4. 本程式由 Python (PyInstaller) 打包，由於未購買數位簽章，下載或執行時可能會被 Windows SmartScreen 或防毒軟體攔截。若出現以下畫面，請點擊 `其他資訊` 並選取 `仍要執行`
